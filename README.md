# AI 3D Choreography Generator (Prototype)

This prototype showcases the product vision for an AI-powered 3D dance choreography generator.
Upload a song, choose a dance style, define the formation, and preview the choreographic brief
that would drive a free, privacy-first 3D rendering pipeline.

## Features in this demo
- Structured briefing form for songs, styles, and formations.
- Teaching-format controls for counts, sections, and breakdowns.
- 2D/3D preview modes with a safe spacing mindset.
- Auto-generated scene plan notes for centers and background dancers.
- Beat grid visualization and sample choreography steps on generate.
- Downloadable preview video export (WebM).
- Suggested next steps for a production-ready pipeline.

## Running locally
```bash
python -m http.server 8000
```
Then open <http://localhost:8000> in your browser.
